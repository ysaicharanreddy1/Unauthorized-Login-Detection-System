# Unauthorized Login Detection System

A role-based authentication system built using Flask.

## Features

- Admin & User login
- 3 password attempt restriction
- Admin security question verification
- Multi-Factor Authentication (OTP via Telegram)
- Real-time Telegram alerts
- Structured incident ID tracking (daily reset format)

## Tech Stack

- Python (Flask)
- HTML + CSS
- Telegram Bot API
- Kali Linux

## How to Run

1. Install dependencies
2. Run: (i)  cd login_ui
        (ii) python3 app.py
4. Open: http://127.0.0.1:5000
